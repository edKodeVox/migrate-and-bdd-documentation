# migrate-and-bdd-documentation

En este repositorio está la documentación para la migración y carga de registros en las bases de datos.

## Archivos

| Archivo | Descripción |
|---|---|
| [`documentacion-migracion-bdd.md`](./documentacion-migracion-bdd.md) | Documentación completa del proceso de migración de BD externa a SQL. Optimizada para Obsidian. |

## Uso con Obsidian

1. Clonar o descargar este repositorio.
2. Abrir **Obsidian** → *Abrir carpeta como vault* → seleccionar la carpeta del repositorio.
3. Editar `documentacion-migracion-bdd.md` completando los campos entre corchetes `[]` con los valores reales del proyecto.
